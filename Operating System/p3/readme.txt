1. Compile and run the program
	1) Execute "make" in terminal to compile the code.
	2) Execute "./diskinfo fileName.img"  in terminal to run the diskinfo program.
	3) Execute ./disklist fileName.img  in terminal to display the content in fileName.img.
	4) Execute "./diskget fileName.img originalFile newFile" in terminal to copy originalFile in fileName.img to newFile at local directory.

2. This code can only handle the file in the root directory.

3. fileName.img should be a img file.
